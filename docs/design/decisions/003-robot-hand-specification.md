# ロボットハンド仕様

## 概要

teleop-hand で制御するロボットハンドの機構・モータ・自由度・マスター-スレーブ構成を決定する。

## 設計判断

### 判断1: 自由度 — 1DOF（指の開閉のみ）

**問題**: teleop-hand で制御するロボットの自由度をどうするか

**選択肢**:
1. 1DOF（指の開閉のみ）
2. ハンド部分のみ（指 + 手首 等）
3. SCARA アーム全体 + ハンド

**決定**: 1DOF（指の開閉のみ）

**理由**:
- finger-tracker の出力は指間距離（1次元量）であり、1DOF の指令値として直接対応する
- まず最小構成で遠隔操作を実現し、段階的に拡張する
- controller_try の `4018_finger` 設定がそのまま使える

**トレードオフ**:
- **利点**: 最小構成で早期に動作確認可能、制御ロジックがシンプル
- **欠点**: 把持対象の位置合わせは手動（アームなし）。対象物への接近はロボット側に人間が置く形

### 判断2: モータ — MTL MDH4018-6750EG03SH

**問題**: 指の開閉に使うモータを何にするか

**決定**: マイクロテック・ラボラトリー MDH4018-6750EG03SH

**仕様**:

| 項目 | 値 |
|------|-----|
| シリーズ | MD-4018（MD-40 シリーズ） |
| 型式 | MDH4018-6750EG03SH（中空軸型） |
| 胴径 | φ40 mm |
| 供給電圧 | DC24V |
| 最大トルク | 1.0 Nm |
| 連続定格トルク | 0.23 Nm |
| 最高回転速度 | 450 rpm |
| 瞬時最大電流 | 10 A |
| 連続定格電流 | 2.3 A |
| トルク定数 | 0.1 Nm/A |
| 線間電機子抵抗 | 1.7 Ω (25°C) |
| エンコーダ | インクリメンタル 6,750 P/R（型番由来、4逓倍で 27,000 P/R） |
| 中空径 | φ6 mm |
| 質量 | 約 0.3 kg |

参照: https://www.mtl.co.jp/motor/motor-1249/

**理由**:
- 研究室の既存資産
- controller_try で制御実績あり（`config/blue_scara/4018_finger.json`）
- 小型・高トルクで指機構に適する

### 判断3: 指機構 — リンク機構（ギア減速比 3:1）

**問題**: モータの回転を指の開閉動作にどう変換するか

**選択肢**:
1. リンク機構
2. ラック＆ピニオン
3. 直動機構（ボールネジ等）

**決定**: リンク機構（ギア減速比 3:1）

**理由**:
- 研究室の既存ハンド機構
- controller_try の `4018_finger.json` に `gear_ratio: 3.0`, `rotate_or_linear: "0"` (回転) として設定済み

**トレードオフ**:
- **利点**: 既存機構を流用、コンパクト
- **欠点**: リンク機構は開閉角度と開き量が非線形な関係になりうる

### 判断4: マスター-スレーブ構成 — カメラ計測（マスター）＋ 電気刺激（力フィードバック）

**問題**: マスター側（人間の手）の計測と力フィードバックをどうするか

**決定**:
- **マスター計測**: finger-tracker（RealSense + YOLOv8 による指間距離計測）
- **力フィードバック**: 電気刺激（将来 Phase 3 で実装）

**構成図**:

```
マスター側（人間の手）           スレーブ側（ロボットハンド）
┌──────────────────┐           ┌──────────────────┐
│ 指サック           │           │ リンク機構指       │
│   ↓               │           │   ↑               │
│ RealSense D435i  │           │ MDH4018 モータ    │
│ + YOLOv8         │           │ + エンコーダ       │
│   ↓               │  UDP      │   ↑               │
│ finger-tracker   │ ──────→  │ teleop-hand       │
│   (Python)       │ 指間距離+3D座標 │   (C++)      │
│                  │           │   ↓               │
│ 電気刺激デバイス ← │ ←──────  │ 反力推定          │
│ （Phase 3）      │  力情報    │ （Phase 3）       │
└──────────────────┘           └──────────────────┘
```

**トレードオフ**:
- **利点**: 非接触計測（指サックのみ）で操作者の手の自由度が高い。電気刺激は小型で装着が容易
- **欠点**: カメラ計測はレイテンシがある（30Hz 程度）。電気刺激は力覚の忠実度が低い

## 関節パラメータ（controller_try から継承）

`config/blue_scara/4018_finger.json` の値:

| パラメータ | 値 | 説明 |
|-----------|-----|------|
| mass | 0.000054 | ロータ慣性モーメント [kg·m²] |
| gear_ratio | 3.0 | ギア減速比 |
| pulse_per_rotation | 6750.0 | エンコーダパルス/回転 |
| multiplication | 4.0 | 4逓倍 |
| force_to_voltage | 10.0 | トルク→電圧変換係数 [V/Nm]。`voltage = f_out * force_to_voltage / gear_ratio` |
| force_limit | 2.0 | 出力軸トルクリミット [Nm]。gear_ratio 除算前に適用。モータ軸換算 = 2.0/3.0 ≈ 0.67 Nm（最大 1.0 Nm 以下で安全） |
| k_p | 10000.0 | 位置ゲイン |
| k_v | 200.0 | 速度ゲイン |
| g_dis | 600.0 | 外乱オブザーバカットオフ [rad/s] |
| g_diff | 600.0 | 擬似微分カットオフ [rad/s] |
| mass_virtual | 0.0000603 | 仮想慣性 |
| damper_virtual | 0.04 | 仮想ダンパ |
| spring_virtual | 0.01 | 仮想バネ |
